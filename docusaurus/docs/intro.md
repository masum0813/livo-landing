# Livo Documentation

This area is the product knowledge base for Livo.

Use it to publish:

- onboarding guides
- setup steps for M3U playlists and Xtream accounts
- daily usage tips
- troubleshooting
- FAQs

## Product scope

Livo is a personal IPTV player. It does not provide content, channels or subscriptions.

## Recommended content structure

Start with these content groups:

- Getting started
- Usage guides
- Troubleshooting
- FAQ

As the app grows, add device-specific guides, release migration notes and feature comparison pages.
