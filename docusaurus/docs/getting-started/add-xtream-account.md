# Add an Xtream account

This guide should explain how users connect an Xtream account to Livo.

## Typical fields

- server URL
- username
- password

## Suggested content

1. Open the Xtream login form.
2. Enter the server URL exactly as provided.
3. Add the username and password.
4. Save and let the app fetch channels and categories.

## Common issues

- incorrect server URL format
- expired credentials
- provider-side downtime
