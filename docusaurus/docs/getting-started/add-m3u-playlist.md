# Add an M3U playlist

Use this page for the exact in-app flow once the UI is finalized.

## Suggested article outline

1. Open the playlist import screen.
2. Choose URL import or file import.
3. Paste the M3U URL or select the file.
4. Save and wait for the playlist to load.

## Troubleshooting notes

- If the URL is invalid, verify it with the provider.
- If categories are missing, the playlist may not include full metadata.
- If content does not play, test another item from the same source.
