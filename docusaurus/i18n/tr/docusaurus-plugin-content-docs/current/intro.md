# Livo Dokumantasyonu

Bu alan, Livo icin urun bilgi merkezi olarak kullanilmalidir.

Burada sunlari yayinlayabilirsiniz:

- baslangic rehberleri
- M3U ve Xtream kurulum adimlari
- gunluk kullanim ipuclari
- sorun giderme yazilari
- sik sorulan sorular

## Urun kapsami

Livo kisisel bir IPTV oynaticidir. Icerik, kanal veya abonelik saglamaz.
