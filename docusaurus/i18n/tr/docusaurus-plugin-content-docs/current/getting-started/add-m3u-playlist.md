# M3U listesi ekle

Bu sayfada uygulama arayuzu netlestiginde gercek akis anlatilmalidir.

1. Liste ice aktarma ekranini acin.
2. URL veya dosya ile ice aktarma secenegini secin.
3. M3U adresini yapistirin veya dosyayi secin.
4. Kaydedin ve listenin yuklenmesini bekleyin.
